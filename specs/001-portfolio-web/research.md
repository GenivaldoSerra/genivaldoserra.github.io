# Research: Portfolio Web

## Technology Choices
- **Decision**: Vanilla HTML5 / CSS3
- **Rationale**: Mandated by the PRD and Constitution to guarantee maximum performance (< 1MB) and simplicity for a static portfolio. No external requests are needed for rendering the core UI.
- **Alternatives considered**: React, Next.js (deferred to future phases as per PRD).

- **Decision**: CSS Grid & Flexbox
- **Rationale**: Essential for mobile-first responsiveness without relying on Bootstrap or Tailwind.

No unknown areas require further clarification. All technical constraints were explicitly predefined in the Constitution and PRD.
